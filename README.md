# GRANJA ABUELO FELIPE - Sitio Web Oficial

## Descripción
Sitio web profesional para las ventas online de productos lácteos de cabra artesanales. Diseñado con una estética moderna y atractiva utilizando colores verde y azul como se especificó.

## Características
- ✅ Diseño responsive y moderno
- ✅ Sección de productos destacados con imágenes atractivas
- ✅ Integración directa con WhatsApp para pedidos
- ✅ Efectos visuales y animaciones interactivas
- ✅ Navegación intuitiva y experiencia de usuario optimizada
- ✅ Paleta de colores coherente (verde y azul)

## Tecnologías Utilizadas
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- Vanta.js (para efectos de fondo animados)
- Three.js (para animaciones 3D)
- Google Fonts (Playfair Display, Inter)

## Estructura del Sitio
```
/
├── index.html          # Página principal con productos
├── nosotros.html       # Página informativa sobre la granja
├── styles.css          # Estilos personalizados adicionales
├── farm-hero.png       # Imagen hero de la granja
├── goat-cheese.png     # Imagen de queso de cabra
├── goat-milk.png       # Imagen de leche de cabra
└── README.md           # Este archivo
```

4. ¡Recibe tus productos frescos en casa!

## Instalación y Uso
1. Descarga todos los archivos en una carpeta
2. Abre `index.html` en tu navegador
3. O despliega en un servidor web local

## Notas de Desarrollo
- El sitio utiliza Tailwind CSS para estilos rápidos y consistentes
- Las animaciones se activan al hacer scroll (fade-in effect)
- Los productos tienen efectos hover para mejor interactividad
- El fondo animado no afecta el rendimiento en dispositivos móviles
